# boutanique
La botanique (du grec βοτανική / botanikê[1], féminin du mot βοτανικός / botanikós, « qui concerne les herbes, les plantes »[2]), nommée auparavant phytologie (du grec φυτόν / phutón, « plante », et λόγος / lógos, « étude »), ou encore biologie végétale, est la science qui a pour objet l'étude des végétaux
